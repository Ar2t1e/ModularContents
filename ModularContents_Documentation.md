# Документация мода ModularContents

Мод **ModularContents** (версия для Forge 1.12.2) предоставляет мощный и расширяемый инструментарий для создания собственных предметов, рецептов, лут-таблиц и кастомных вкладок креатива (Creative Tabs) без необходимости писать код на Java. Всё настраивается с помощью удобных JSON-файлов, а также визуального редактора внутри самой игры.

В этом документе описаны все новые функции и системы мода.

---

## 1. Система Аирдропов (Airdrops)

Система аирдропов позволяет сбрасывать контейнеры с лутом (парашютные посылки) с небес. Лут внутри генерируется полностью на основе кастомных JSON-таблиц. 

### 1.1 Как вызвать Аирдроп

Существует 4 основных способа вызова аирдропа:

1.  **Глобальные Аирдропы (Global Airdrops):**
    Сервер сам может автоматически сбрасывать аирдропы случайному игроку на сервере раз в несколько дней.
    *Настройка:* Выполняется в файле `modularcontents_settings.cfg` в корневой папке `ModularContents/`. Вы можете настроить интервал в днях, минимальный/максимальный радиус от игрока и разрешенные таблицы лута.
2.  **Сигнальная Ракета (Signal Flare):**
    Бросаемый предмет. При падении на землю начинает выпускать цветной дым. Запускается скрытый таймер (от 10 до 50 секунд), по истечении которого в небе над ракетой спавнится Аирдроп. Когда он приземлится, игрок получит уведомление в чат с направлением и дистанцией до груза.
3.  **Рация (Radio):**
    Предмет с прочностью (10 использований). При использовании (ПКМ) Аирдроп моментально вызывается в случайном месте (в радиусе от 50 до 200 блоков). Точные координаты и время до падения (ETA) сразу выводятся игроку в чат. Аирдроп "зависает" в небе и начнет падать только спустя случайную задержку (10-50 секунд).
4.  **Ноутбук (Tactical Laptop):**
    Устанавливаемый блок. При клике ПКМ открывается полноэкранный интерфейс тактической миникарты. Игрок видит рельеф местности (лес, вода, горы). С помощью колесика мыши можно масштабировать (зуммировать) карту. Игрок может кликнуть на любую точку в радиусе 128 блоков, нажать "Call Airdrop" и вызвать посылку ровно в эту точку.
5.  **Команда Администратора:**
    `/mc airdrop <x> <z> [имя_таблицы]` — спавнит аирдроп в указанных координатах. Опционально можно передать название конкретной лут-таблицы (например, `military_loot`).

### 1.2 Папки Лут-Таблиц

Файлы таблиц лута хранятся в папке ваших контент-паков:
`[Папка_Игры]/ModularContents/[Ваш_Пак]/loot_tables/airdrops/*.json`

Пример структуры JSON (вы также можете создавать их визуально через внутриигровой интерфейс):
```json
{
  "weight": 50,
  "items": [
    {
      "item": "minecraft:golden_apple",
      "meta": 0,
      "min": 1,
      "max": 2,
      "chance": 0.3
    }
  ]
}
```

## 2. Внутриигровой Генератор Контента (Content Creator)

Теперь вам больше не нужно писать JSON-файлы вручную. Мод включает в себя полноценный графический интерфейс для визуального создания контента прямо во время игры.

### 2.1 Как открыть
Находясь в **Творческом режиме (Creative Mode)**, нажмите клавишу **F8** (клавишу можно изменить в настройках управления Minecraft).

### 2.2 Вкладка "Loot" (Генератор таблиц лута)
*   В центре экрана находится инвентарь на 27 слотов (как у обычного сундука).
*   Поместите в слоты любые предметы, которые вы хотите добавить в Аирдроп.
*   **Выделение слота:** Кликните ЛКМ по любому предмету в сетке. Он выделится оранжевой рамкой, и справа откроется панель настроек для *конкретно этого предмета*.
*   На правой панели вы можете тонко настроить:
    *   **Min**: Минимальное количество этого предмета.
    *   **Max**: Максимальное количество (по умолчанию берется размер стака предмета, который вы положили).
    *   **Chance**: Шанс появления предмета (от `0.0` до `1.0`, где `0.5` = 50%).
*   Слева вверху укажите **File Name** (например, `military_loot`) и **Weight** (вес таблицы для случайного выбора).
*   Нажмите **Generate JSON**. Файл моментально появится в папке `ModularContents/generated/loot_tables/airdrops/`.

### 2.3 Вкладка "Items" (Создание предметов)
*   Заполните поля: `Item ID` (техническое имя), `Display Name` (название в игре), `Max Stack` (размер стака), `Tab` (вкладка креатива) и `Durability` (прочность, 0 = бесконечный).
*   Нажмите **Generate JSON** для создания файла предмета.

### 2.4 Вкладка "Recipes" (Создание рецептов)
*   **Outputs (Красный фон):** Положите в верхний ряд предметы, которые должны получаться при крафте.
*   **Inputs (Серый фон):** Положите в два нижних ряда предметы, которые требуются для крафта. 
    *(Размер стака автоматически учитывается как нужное количество!)*
*   Настройте категорию, время крафта (в тиках) и минимальное количество дропа.
*   Нажмите **Generate JSON**.

### 2.5 Вкладка "Tabs" (Создание вкладок креатива)
*   Позволяет создавать свои собственные вкладки в креативном инвентаре (Creative Tabs).
*   Укажите ID вкладки, её красивое название и ID предмета-иконки (например, `minecraft:diamond_sword`).
*   Сгенерируйте файл. После этого вы сможете указывать ID этой вкладки при создании новых предметов на вкладке "Items".

## 3. Динамические предметы для Аирдропов (Custom Airdrop Callers)

Помимо встроенных Рации и Сигнальной ракеты, вы можете создавать **свои собственные предметы для вызова Аирдропов**.

Во время создания предмета через JSON в папке `items/`, добавьте блок `airdrop_caller`:
```json
{
  "id": "vip_caller",
  "display_name": "VIP Supply Drop",
  "airdrop_caller": {
    "is_flare": false,
    "delay_min": 100,
    "delay_max": 200,
    "distance_min": 10.0,
    "distance_max": 50.0,
    "loot_table": "military_loot",
    "consume_on_use": true
  }
}
```
*   `is_flare`: Если `true`, предмет кидается как граната (как сигнальная ракета). Если `false` — используется сразу в руке (как рация).
*   `loot_table`: Если указано, предмет будет призывать строго определенную таблицу лута. Оставьте `""` для рандома.

## 4. Команда перезагрузки

Вам не нужно перезапускать клиент или сервер, чтобы применить изменения.
Введите команду `/mc reload` в чат (требуются права администратора / OP). Мод мгновенно пересканирует все ваши контент-паки и загрузит новые/измененные рецепты и таблицы лута прямо "на горячую".
